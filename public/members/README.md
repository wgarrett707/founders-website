# Team Member Photos

Place team member photos in this directory.

## File Naming
Name your photos with descriptive filenames (e.g., `john-doe.jpg`, `jane-smith.png`).

## Supported Formats
- JPEG (.jpg, .jpeg)
- PNG (.png)
- WebP (.webp)

## Usage
Update the `team-members.csv` file in the root directory with the path to each member's photo. For example:
- `public/members/john-doe.jpg`
- `public/members/jane-smith.png`

## Notes
- Photos will be automatically resized to fit the team member cards
- If a photo fails to load, a placeholder will be shown
- Recommended aspect ratio: 3:4 (portrait orientation)
